Китайское видение ИИ — это интеллектуальные системы, подключенные к огромным промышленным экосистемам,

которые позволяют производить более дешевые товары для всех + модели с открытым исходным кодом, которые доступны бесплатно по всему миру по низкой цене для всех.

Американское видение — это несколько олигархических компаний, которые владеют всеми базовыми моделями,

а затем используют их, чтобы уничтожить всю реальную экономику, при этом награждая граждан бесполезным базовым доходом и заманивая их в огромную экономику отчаяния.

Мы больше не являемся хорошими парнями, мои соотечественники,

мы заслуживаем проиграть битву за ИИ, мы не имеем права поучать мир, когда на этот раз мы явно являемся злодеями. https://x.com/okaythenfuture/status/2026250944112132135?s=20

Есть три момента.

  1. Китайские модели бесплатны для всех.

  2. Уже сейчас оборудование, которое позволяет пускать дома модель уровня GPT-4/5 (дистиллированный дипсик на 70B параметров), стоит 2-3 тысячи долларов.

  3. Я думаю через пару лет прогресс в алгоритмах и железе сделает GPT-5 практичным дома. Уолл Стрит может сказать своим влажным мечтам досвиданья.
