Короче, делать будем вот как.

В связи с засилием китайских моделей, которые якобы должны работать на гражданском оборудовании, мы с @maximgavrilov решили проверить, насколько это все пригодно для офицеров (маленькая стрелка вниз, большая стрелка вверх).

Брать решили для теста Макмини в топовой конфигурации. 64GB RAM, 1TB SSD, 2400 USD. 64Гб, предположительно, должно быть достаточно для каждого!

UPD: Нет! Mac Studio 128GB RAM, 1Tb SSD, 3700 USD. Чтоб наверняка. Я думаю, таким станет мак мини через пару лет, и оно вернется в 2-3 тысячи.

Проверим, насколько оно сравнимо с языковыми моделями, которые продаются за 20 баксов онлайн. И может ли это нормальный человек самостоятельно настроить?

На это есть две недели, окажется бесполезным — вернем. Тестировать буду не только я, но и моя жена, которая использует языковые модели для подготовки занятий по музыке для детей.

О том, как пройдет, напишем.

Агентные агенты, аргл-баргл! 🍿
