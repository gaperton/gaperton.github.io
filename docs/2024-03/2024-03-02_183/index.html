В общем, такое.

Ставить надо Linux Mint. Не забыть кликнуть галку про проприетарные кодеки.

Потом, поставить проприетарные драйверы, прокликав через гуй (оно само предложит)

Дальше, зайти на сайт Brave, и поставить оттуда по инструкции Brave (так лучше чем через гуй). После чего снести файрфокс.

Дальше, зайти на сайт Телеги, и поставить телегу через бинарь (инструкцию куда копировать ищем отдельно). Потому что если ставить через гуй то оно тупо не работает ваще. По крайней мере, у меня.

Последнее что надо сделать — поставить майкрософтские TrueType шрифты, и шрифт Noto, который надо потом выбрать как дефолт. Это если у вас кровоточат глаза от шрифта Ubuntu, который стоит по умолчанию.

vs code надо ставить тоже не через гуй, а из пакета на сайте. Потому, что иначе оно нормально не работает.

Для синхронизации файлов между компами без участия облака (я знаю, вы этого хотите) — https://www.resilio.com/individuals/.

Для заметок надо ставить Obsidian (это уже можно через гуй). Эта штука хранит все в маркдаун, и прекрасно синхронизирует заметки через Resilio.

И оно, в принципе, вполне терпимо и удобно. На уровне.

Как вы наверное уже понимаете, этот стек софта специально придуман, чтобы работать мимо Shitbag Corporations, не особенно при этом страдая.
