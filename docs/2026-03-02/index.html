35b-a3b это близко к минимуму того что вообще можно терпеть, как по скорости так и по качеству.

❗️️Всем кто страдал от низкой скорости Claude Code на локальных моделях

Видите файл settings.json VS Code? Добавьте там CLAUDE_CODE_ATTRIBUTION_HEADER, как написано. Это надо Claude Code такую переменную среды, чтоб не тормозил.

И наслаждайтесь нормальной скоростью обработки промпта. На всякие странные штуки типа Pi Code переходить не надо.

Выбрасывать свой M4 Max (или что у вас там тормозит) — тоже. Просто не MLX надо использовать, а GGUF. Он по токенам медленнее, зато промпт умеет кэшировать.

U r welcome.

``{
"git.enableSmartCommit": true,
"git.confirmSync": false,
"git.autofetch": true,
"claudeCode.preferredLocation": "panel",
"editor.minimap.enabled": false,
"claudeCode.environmentVariables": [
{
"name":"CLAUDE_CODE_ATTRIBUTION_HEADER",
"value": "0"
},
{
"name": "ANTHROPIC_BASE_URL",
"value": "http://localhost:1234"
},
{
"name": "ANTHROPIC_AUTH_TOKEN",
