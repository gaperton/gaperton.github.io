**Лучший способ работать с удаленным сервером по ssh**

Это Visual Studio Code, если поставить на него extension от Microsoft Remote Development. С ним вы можете присоединиться к любому серверу по ssh. И у вас будет все. И терминал, и файловый браузер, и редактор. Все это будет быстро работать, и вам не нужен удаленный доступ к экрану.

Хозяйке на заметку. Вот так мы делаем себе домашний Ubuntu server и работаем с ним, не особо страдая.

Зачем мы делаем себе Ubuntu Server? Ну, конечно же, для того, чтобы запускать на нем языковые модели! Мы ведь не хотим, чтобы UI отжирал лишние 2 гигабайта памяти, которые могут пойти на контекст.

Да, все плагины Visual Studio Code, разумеется, тоже с этим работают, включая Claude Code и Codex. Вы можете админить ваш Ubuntu сервер при помощи Claude Code. Не надо ничего знать. Пусть он красноглазит.

Хотя я, честно говоря, предпочитаю на Ubuntu сервер [Hermes Agent](https://hermes-agent.nousresearch.com/). Он у меня админит все мои компьютеры.
