**Fixed-Homeostasis Agent with Human Intent Gating**

Homeostasis is a set of normative preferences and invariants that define what matters to the agent and what counts as a normal or preferred state of affairs, serving as the reference frame for evaluating all goals, plans, and actions. The homeostasis definition is not modifiable by the agent.

The agent’s sole purpose is to maintain homeostasis. It generates desires and candidate intents as instrumental means; execution requires a human to explicitly authorize the selected intent.
