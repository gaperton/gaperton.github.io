Правка нотации. Надо писать и вопрос, и хотя бы главные семантические роли — объект, субъект/агент, адресат, модальность. Вопрос можно опустить если есть роль. И стрелку.

``[кто/subj: [какая: неродная] дочка
  [кого/poss:->] сестры
  [кого/poss:->] жены
  [кого/poss:->] брата
  [кого/poss:->] Серёги Молина`]`

Минимальный список ролей (канон)

subj / кто — субъект клаузы; при необходимости уточнение семантики: [(sem: agent|experiencer…)].

obj / что|кого — прямой объект/пациент/тема.

iobj / кому — адресат/получатель (дательный, «кому»).

mod / — — локальный модальный носитель (debeo, licet, можно, must…); итоговая рамка — один раз как [(μ: D|P|A|◇|□|…)].

poss / чей — владение; строит генитивные цепочки (часто со стрелкой).

loc / где — место; src / откуда — источник; goal / куда — цель пути.
(Трио «путь/локатив»; самые частые обстоятельства.)

tmp / когда — время (включая «сначала/затем»; для порядка можно добавить τ-метки).

mnr / как — способ/образ действия (наречия, предлоги).

instr / чем — инструмент/средство.

cause / почему и purp / зачем — причина и цель (часты в учебных текстах).

Всё остальное — расширения по мере необходимости: topic/о-чём, cond/если, concess/несмотря-на, result/итог, extent/степень и т. п. Они маппятся на UD/SRL, но для старта не обязательны.
